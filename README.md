## üöÄ Project Workflow
This project follows a professional data lifecycle:
1.  **Data Cleaning & EDA (Python/Jupyter):** Handled missing values, outliers, and data type normalization.
2.  **Database Integration (PostgreSQL):** Exported cleaned data into pgAdmin4 for structured querying.
3.  **Business Logic (SQL):** Authored complex queries to extract key performance indicators (KPIs).
4.  **Data Visualization (Power BI):** Connected the live SQL database to Power BI to build a dynamic dashboard.
5.  **Reporting:** Synthesized technical findings into a stakeholder-ready presentation.
